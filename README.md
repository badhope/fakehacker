## 🎛️ 界面折叠控制

### 什么是折叠功能？

折叠功能允许您隐藏暂时不需要的界面区域，让主显示区获得更大空间。所有折叠状态会自动保存，下次打开时自动恢复。

### 可折叠的区域

| 区域 | 位置 | 说明 |
|------|------|------|
| **顶部导航栏** | 屏幕顶部 | 显示系统状态、加密级别、运行时间等 |
| **左侧按钮区** | 屏幕左侧 | 26 个字母功能按钮 |
| **右侧控制面板** | 屏幕右侧 | 高级输入、快捷操作、系统监控 |
| **底部信息栏** | 屏幕底部 | 显示时间、延迟、带宽等信息 |

### 如何折叠/展开界面元素

#### 方法一：使用折叠按钮（推荐新手）

每个可折叠区域都有一个折叠控制按钮：

1. **顶部导航栏** 
   - 按钮位置：状态栏右侧
   - 按钮图标：`▲`
   - 操作：点击按钮即可折叠/展开

2. **左侧按钮区**
   - 按钮位置：左侧面板标题栏右侧
   - 按钮图标：`◀`
   - 操作：点击按钮即可折叠/展开

3. **右侧控制面板**
   - 按钮位置：右侧面板标题栏右侧
   - 按钮图标：`▶`
   - 操作：点击按钮即可折叠/展开

4. **底部信息栏**
   - 按钮位置：底部信息栏右侧
   - 按钮图标：`▼`
   - 操作：点击按钮即可折叠/展开

#### 方法二：使用键盘快捷键（推荐高级用户）

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **T** | 切换顶部导航栏 | Toggle Top bar |
| **L** | 切换左侧按钮区 | Toggle Left panel |
| **R** | 切换右侧控制面板 | Toggle Right panel |
| **B** | 切换底部信息栏 | Toggle Bottom bar |
| **F** | 切换所有区域 | 一键进入/退出全屏模式 |
| **H** | 显示帮助信息 | 打开折叠控制帮助面板 |

**使用技巧：**
- 快捷键在输入框外任意位置都有效
- 在输入框中打字时，快捷键会自动禁用，避免误操作
- 折叠按钮的图标会在折叠后自动翻转，提示当前状态

## ⌨️ 键盘快捷键

### 界面控制快捷键

| 按键 | 功能 | 使用场景 |
|------|------|----------|
| `T` | 折叠/展开顶部状态栏 | 需要更多垂直空间时 |

| 按键 | 功能 | 说明 |

| `Esc` | 取消/关闭面板 | 关闭打开的对话框 |

### 组合快捷键（部分浏览器支持）

| 组合键 | 功能 |
|--------|------|
| `Ctrl + T` | 新标签页（浏览器默认） |

| `F11` | 浏览器全屏（浏览器默认） |

## 🖱️ 基本操作说明

### 左侧按钮区操作

**26 个字母按钮功能分类：**

1. **扫描类（绿色边框）**
   - 功能：网络扫描、系统探测
   - 示例：A(网络扫描)、S(端口扫描)

2. **破解类（红色边框）**
   - 功能：密码破解、系统入侵
   - 示例：D(暴力破解)、F(漏洞利用)

3. **工具类（黄色边框）**
   - 功能：系统工具、辅助功能
   - 示例：G(生成密码)、H(查看日志)

4. **黑客类（绿色边框）**
   - 功能：高级黑客操作
   - 示例：J(注入攻击)、K(键盘记录)

**操作方式：**
- 鼠标点击：直接点击字母按钮
- 键盘操作：按下对应字母键

### 中间显示区操作

**日志输出区：**
- 显示所有操作的详细日志
- 自动滚动到最新日志
- 不同颜色代表不同类型：
  - 🟢 绿色：成功/正常操作
  - 🔵 蓝色：信息提示
  - 🟡 黄色：警告信息
  - 🔴 红色：错误/危险操作

**数据可视化区：**
- 网络流量图
- 系统资源监控
- 攻击进度显示

### 右侧控制面板操作

**高级输入区：**
```
DECRYPTION KEY:
[输入框]
```
- 用于输入高级指令
- 支持数字 1-9 触发特殊功能
- 输入完成后按 Enter 执行

**快捷操作按钮：**

| 按钮 | 功能 | 效果 |
|------|------|------|
| PANIC BUTTON | 紧急按钮 | 立即停止所有操作，清除痕迹 |
| GLITCH MODE | 故障模式 | 启用视觉故障效果 |
| REBOOT SYSTEM | 重启系统 | 重置所有状态 |
| CHANGE THEME | 切换主题 | 循环切换不同主题 |
| TOGGLE AUDIO | 切换音效 | 开启/关闭音效 |

**系统监控：**
- THREADS: 当前运行线程数
- PACKETS: 数据包数量
- ERRORS: 错误计数

### 底部信息栏操作

**显示信息：**
- 系统状态指示灯（绿色闪烁）
- 当前时间
- 网络延迟
- 带宽速度

**提示信息：**
- 操作提示
- 系统通知
- 成就解锁提醒

## � 自动保存功能

### 什么会被保存？

- ✅ 折叠状态（哪些区域被折叠）
- ✅ 主题选择
- ✅ 音效设置
- ✅ 游戏进度（成就、任务）
- ✅ 统计数据

### 保存位置

所有数据保存在浏览器的 LocalStorage 中，无需担心丢失。

### 清除数据

如需重置所有数据：
1. 打开浏览器开发者工具（F12）
2. 进入 Application/Storage 选项卡
3. 清除 LocalStorage

## ❓ 常见问题

### Q1: 折叠后找不到展开按钮？
**A:** 折叠按钮会跟随区域一起隐藏。使用键盘快捷键即可重新展开：
- 按 `T` 展开顶部
- 按 `L` 展开左侧
- 按 `R` 展开右侧
- 按 `B` 展开底部
- 按 `F` 全部展开

### Q2: 快捷键不响应？
**A:** 检查以下几点：
1. 确保没有在输入框中打字
2. 确保浏览器窗口处于焦点状态
3. 检查是否有浏览器扩展冲突
4. 尝试刷新页面（Ctrl+R）

### Q3: 折叠状态没有保存？
**A:** 可能原因：
1. 浏览器禁用了 LocalStorage
2. 使用了隐私/无痕模式
3. 浏览器缓存已满
解决方法：清除浏览器缓存并重启浏览器

### Q4: 移动端可以使用吗？
**A:** 可以！移动端优化：
- 默认自动折叠侧边栏
- 支持触摸操作
- 响应式布局适配

### Q5: 如何完全退出全屏模式？
**A:** 三种方法：
1. 按 `F` 键切换回普通模式
2. 按 `F11` 退出浏览器全屏
3. 按 `Esc` 键（部分浏览器）

### Q6: 帮助面板怎么关闭？
**A:** 两种方法：
1. 点击右上角的 "✕ 关闭" 按钮
2. 再次按 `H` 键
3. 点击帮助面板外的区域
