## ⚡ 快速开始
### 1. 克隆仓库（假装你在拉取「核心漏洞库」）
```bash
git clone https://github.com/badhope/fakehacker.git
cd fakehacker
```

### 2. 运行脚本（开始你的奥斯卡级表演）
假设核心脚本是 Python 版（如果是其他语言，替换对应命令即可）：
```bash
# 先装依赖（如果有的话，没有就跳过）
pip install -r requirements.txt

# 启动装逼模式
python fakehacker.py
```

### 3. 效果炸裂
运行后，你的终端会疯狂刷屏各种「专业术语」：
```

[+] 突破目标防火墙防护...
[+] 解密 RSA 2048 位密钥...
[+] 获取管理员权限成功！
[+] 正在写入后门程序...
[!] 警告：检测到反黑客系统，正在伪装IP...
[✓] 操作完成：目标系统已「被控」（假的）
```
朋友看了直接喊「大佬」，问你能不能帮他盗QQ号的程度🤣

## 🎨 自定义装逼姿势
觉得默认话术不够劲？改！
1. 打开 `config.py`（或 `config.json`）
2. 修改 `hack_texts` 列表，替换成你想要的装逼文案：
   ```python
   hack_texts = [
       "[+] 正在破解王者荣耀全皮肤权限...",
       "[+] 嗅探到隔壁老王的WiFi密码：12345678...",
       "[+] 绕过微信支付风控系统...（纯装X，别真试）",
   ]
   ```
3. 还能改刷屏速度、颜色、字体，怎么帅怎么来～
