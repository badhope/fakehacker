# Visual Effects Lab - 简单版 (v3.0)

这是一个**超级简单**的 HTML5 Canvas 特效展示项目，只有一个 HTML 文件，没有任何复杂的依赖。

## 🎯 特性

- ✅ **单文件**：只有一个 `index.html`，打开即用
- ✅ **零依赖**：不需要 Node.js、npm 或任何构建工具
- ✅ **4 种特效**：粒子流动、波浪效果、星空闪烁、矩阵雨
- ✅ **响应式**：完美支持桌面端和移动端
- ✅ **可调节**：速度、数量都可以实时调整

## 🚀 快速开始

### 方法 1：直接打开
直接双击 `index.html` 文件即可在浏览器中运行

### 方法 2：本地服务器（推荐）
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server -p 8000
```

然后访问 `http://localhost:8000`

### 方法 3：GitHub Pages
访问：https://badhope.github.io/fakehacker/v3/

## 🎮 操作说明

### 鼠标操作
- **拖动滑块**：调整速度和数量
- **点击按钮**：暂停/继续、重置、全屏
- **下拉菜单**：选择不同特效
- **双击 Canvas**：切换下一个特效

### 键盘快捷键
- **空格键**：暂停/继续
- **F 键**：全屏切换

## 🎨 特效列表

### 1. 粒子流动 (Particles)
彩色粒子在屏幕上随机移动，碰到边界会反弹

### 2. 波浪效果 (Waves)
5 条彩色正弦波在屏幕上波动

### 3. 星空闪烁 (Stars)
200 颗星星在黑色背景上闪烁

### 4. 矩阵雨 (Matrix)
绿色字符像电影《黑客帝国》一样下落

## 📱 移动端支持

- 自动适配手机、平板、桌面
- 触摸友好的按钮大小
- 响应式布局
