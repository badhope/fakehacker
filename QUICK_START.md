# 🚀 量子黑客终端 - 快速上手指南

## 📌 5 分钟快速开始

### 第一步：打开应用
1. 访问 http://localhost:8000
2. 等待启动动画完成（约 2 秒）
3. 看到主界面

### 第二步：体验基本功能
1. **点击任意字母按钮**（如 A、S、D）
2. 观察中间日志区的输出
3. 查看数据可视化区的变化

### 第三步：尝试折叠功能
1. 按 **T** 键 - 折叠顶部导航栏
2. 按 **L** 键 - 折叠左侧按钮区
3. 按 **R** 键 - 折叠右侧控制面板
4. 按 **B** 键 - 折叠底部信息栏
5. 按 **F** 键 - 一键全屏/恢复

### 第四步：查看帮助
- 按 **H** 键打开帮助面板
- 查看所有快捷键说明
- 点击右上角"✕ 关闭"或再次按 H 关闭

## 🎯 核心快捷键（必记）

| 按键 | 功能 | 记忆方法 |
|------|------|----------|
| **T** | **T**op - 顶部导航栏 | Top 的首字母 |
| **L** | **L**eft - 左侧按钮区 | Left 的首字母 |
| **R** | **R**ight - 右侧控制面板 | Right 的首字母 |
| **B** | **B**ottom - 底部信息栏 | Bottom 的首字母 |
| **F** | **F**ullscreen - 全屏模式 | Fullscreen 的首字母 |
| **H** | **H**elp - 帮助 | Help 的首字母 |

## 🎮 三种常用布局

### 布局 1：新手模式（默认）
```
所有区域都展开
适合：第一次使用，熟悉各个功能
```

### 布局 2：专注模式
```
折叠：顶部 + 底部 + 右侧
保留：左侧按钮区 + 中间显示区
快捷键：依次按 T、B、R
适合：专注查看日志和操作
```

### 布局 3：全屏模式
```
折叠：所有区域
保留：仅中间显示区
快捷键：按 F
适合：沉浸式体验、演示
```

## 💡 实用技巧

### 技巧 1：快速隐藏/显示
- 想全屏查看日志？按 **F**
- 想恢复完整界面？再按 **F**

### 技巧 2：快捷键连招
```
T → L → R → B = 全部折叠
F = 一键搞定
```

### 技巧 3：避免误操作
- 在输入框打字时，快捷键自动禁用
- 不会意外折叠界面

### 技巧 4：状态保存
- 折叠状态自动保存
- 刷新页面后自动恢复
- 无需手动保存

## ❓ 常见问题速查

### Q: 折叠后找不到展开按钮？
**A**: 按对应的快捷键：
- 顶部看不见？按 **T**
- 左侧看不见？按 **L**
- 右侧看不见？按 **R**
- 底部看不见？按 **B**
- 都不知道？按 **F**

### Q: 快捷键没反应？
**A**: 检查：
1. 是否在输入框中？（在输入框会禁用快捷键）
2. 浏览器窗口是否激活？
3. 刷新页面试试（Ctrl+R）

### Q: 如何重置所有设置？
**A**: 
1. 按 F12 打开开发者工具
2. 进入 Application/Storage
3. 清除 LocalStorage
